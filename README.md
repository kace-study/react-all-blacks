## Set Up in Your Local
you can run the following command:
```
git clone git@github.com:kace-study/react-all-blacks.git
npm start
```

## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    compornents/
    App.css
    App.js
    index.css
    index.js
```
